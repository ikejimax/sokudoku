# 📚 速読解トレーニング Web Edition

ソフトウェア設計の授業で作成した作品です。開発期間は1週間の指定があり、その中で設計・コーディング・検証を含めて開発しました。
開発の流れは、アプリの設計でアプリに必要な機能や画面遷移などを定義したのちに、コーディングは「google Antigraviti」で大枠を作成、「claude」で調整を行いました。その後、自身でコードレビューを行い、微調整を行いました。
日本語の速読力を体系的に鍛えるWebアプリです。  
眼筋運動から理解度クイズまで、速読に必要なスキルを10種類以上のトレーニングでカバーしています。

## 公開URL

https://d351bkpacwlppd.cloudfront.net

## 機能一覧

### フルトレーニング（推奨）
全フェーズを通して速読力を底上げするコース（約20〜25分）

### 個別トレーニング
| トレーニング | 内容 |
|---|---|
| 📏 読書速度計測 | 現在のWPM（文字/分）を計測 |
| 👁 眼筋運動 | 7種類のパターンで眼筋を多角的に鍛える |
| 🔢 シュルテテーブル | 5×5マスで1〜25を順番にタップ。眼球運動と周辺視野を同時に鍛える |
| 🔵 周辺視野フラッシュ | 中央を見つめたまま数字を認知する脳トレ型トレーニング |
| ⚡ フラッシュ読書 | 文節→ブロック→ページの3段階RSVPトレーニング |
| 🔤 単語フラッシュ | 重要キーワードを高速フラッシュし意味の定着を促す |
| 🧩 スパン測定 | 一瞬で認識できる文字数（視野スパン）を測定 |
| 💡 ガイド読み | ハイライト追従で視野を広げる |
| 🧠 右脳イメージ変換 | キーワードから場面を連想し右脳を活性化 |
| ❓ 理解度クイズ | 3択で読んだ内容の理解度を確認 |

### その他の機能
- ユーザー登録・ログイン機能（パスワードはSHA-256でハッシュ化）
- 学習履歴の保存とWPM推移の確認
- 横書き・縦書きモードの切り替え
- ダークモード対応

## 使用技術

- HTML / CSS / JavaScript（フロントエンドのみ・サーバーレス構成）
- AWS S3（静的ファイルホスティング）
- AWS CloudFront（HTTPS配信・CDN）

## システム構成

```
ユーザー → CloudFront（HTTPS）→ S3（index.html / style.css / app.js）
```

授業の範疇での作品で開発期間が短かったため、DBなどは実装せず。データはすべてブラウザのlocalStorageに保存されます。サーバーへの通信は発生しません。

## 工夫した点

- サーバーを使わずS3+CloudFrontだけで本番運用可能な構成にした
- パスワードをWeb Crypto API（SHA-256）でハッシュ化し、平文保存を避けた
- 縦書き・横書きの切り替えやダークモードなど、UIの使いやすさを意識して設計した
